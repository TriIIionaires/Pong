A pong game application developed using the .NET Framework where players can play pong against each other. The project was made to learn how to manage server and client communication while still creating a fun and timeless game.

## Technologies
[![My Skills](https://skillicons.dev/icons?i=cs,html,css,dotnet,bootstrap,visualstudio&theme=light)](https://skillicons.dev)  
**Including:** SignalR to manage server-client communication

## Features
- **Local Multiplayer:** Players on the server's network can play with each other.
- **Player Queue:** Players can wait in a queue until another player joins to be matched into a game.
- **Paddle Movement:** Use the up and down arrow keys to move the paddle.
- **First to 5 Points:** The player who reaches 5 points first wins the game.

## Running the Project
To run the project in your local environment, follow these steps:
1. Clone the repository to your local machine into Visual Studio.
2. Repeat step one for how many players you want to play onto their own respective machine.
3. Have one machine fire up the PongAPI.
4. Run PongMAUI on all machines and ensure they are all on the same network.
5. Play the game.
